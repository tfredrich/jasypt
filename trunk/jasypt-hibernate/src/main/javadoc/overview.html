This document is the API specification for the Jasypt Java Simplified Encryption library integration package for Hibernate, version 1.2.

You can have a look at the hibernate type class here: {@link org.jasypt.hibernate.EncryptedTextType}.

If you are using the Spring Framework or some other IoC container, you may be interested in: {@link org.jasypt.hibernate.HibernatePBEEncryptor}.

If you are not using Spring nor any other IoC container, you may be interested in: {@link org.jasypt.hibernate.HibernatePBEEncryptorRegistry}.
